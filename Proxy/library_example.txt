System.Reflection.DispatchProxy

Это встроенная библиотека .NET для создания динамических прокси-объектов во время выполнения. Она позволяет перехватывать вызовы методов через механизм рефлексии, предоставляет механизм для создания экземпляров прокси-объектов и обработки их диспетчеризации методов.
Особенность: требует целевой объект (работает через агрегирование)
Пример использования: реализовывание автоматического кэширования результатов методов сервиса без изменения исходного кода сервиса.
